# QGIS Template Generator

## Goal
Auto-create a ready-to-use `.qgz` QGIS project from a packaged folder generated by `prepare_qgis_package.py`.

## Step 1: Build package
```powershell
Backend/.venv311/Scripts/python -m Backend.sea_level_risk.qgis.prepare_qgis_package \
  --city honolulu \
  --dem data/honolulu_dem.tif \
  --realtime-dir Backend/sea_level_risk/outputs/realtime/honolulu \
  --out-root Backend/sea_level_risk/outputs/qgis_packages
```

## Step 2: Create QGIS template project
Open QGIS -> `Plugins` -> `Python Console`, then run:
```python
import sys
sys.path.append(r"D:\CODE\Projects\Sea_Level_Rise - New ver")

from Backend.sea_level_risk.qgis.create_qgis_template import create_project

pkg = r"D:\CODE\Projects\Sea_Level_Rise - New ver\Backend\sea_level_risk\outputs\qgis_packages\honolulu_20260308_184339"
out_qgz = create_project(pkg)
print(out_qgz)
```

This will generate:
- `anti_flood_template.qgz` inside the package folder.

## What the template includes
- Project CRS: `EPSG:4326`
- Layer order: `dem -> flood_20 -> flood_50 -> flood_100 -> hotspots`
- Applied `.qml` styles from `styles/`

## Notes
- Run the script inside QGIS Python Console (PyQGIS required).
- If layer paths change, rebuild package and regenerate template.
